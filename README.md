# Website shops online
## Cộng nghệ sử dụng
  - PHP
  - Bootstrap

## Một số chức năng chính
  ### Admin
    * Quản lý đơn hàng.
    * Quản lý sản phẩm.
    * Quản lý bình luận.
    * Quản lý tài khoản.
    * Thống kê sản phẩm.
    * Quản lý đơn hàng.
  ### Client
    * Quản lý thông tin tài khoản.
    * Đăng ký tài khoản.
    * Mua hàng.
    * Thanh toán.
    * Xem lịch sử mua hàng.
